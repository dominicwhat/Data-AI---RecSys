# 資料人才種子計畫

## 當初對於課程的期待、回顧期待的達成程度

當初會想上課程的原因，是因為最近要進行推薦系統的研究，而推薦系統的研究，本質上的應用就是商業場景。
而資料人才種子計畫，就是以具體的商業場景為背景，介紹推薦系統在商業場景的使用，讓我更了解文獻上研究問題的重要性，並對研究上的發想，有了很大的啟發。
